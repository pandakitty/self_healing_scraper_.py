# Area Code Geo Mapper & Data Sentinel

A high-precision geospatial and automation pipeline designed for socio-economic analysis and self-healing data extraction.

## Core Capabilities
* **Geospatial Sentinel:** Maps demographic trends by performing spatial joins between dissolved NPA boundaries and Census/ACS data.
* **Self-Healing Scraper:** A diagnostic 'Sentinel' class that monitors data integrity and flags structural changes in web sources.
* **Institutional Rigor:** Standardized to EPSG:4326 with $3\sigma$ anomaly detection for forensic-grade data registries.

## Technical Stack
* **Languages:** Python 3.11
* **Geospatial:** GeoPandas, Shapely, Fiona
* **Automation:** BeautifulSoup, Requests, Pandas

## Professional Application
This repository serves as a technical showcase for **Institutional Registry Architecture**, prioritizing data provenance and high-fidelity visual contrast. Engineered for roles requiring 'Technical Finishing'—from refactoring legacy scripts to deploying production-ready documentation.
